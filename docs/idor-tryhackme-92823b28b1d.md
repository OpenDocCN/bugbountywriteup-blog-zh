> **在编码的 ID 中查找 IDOR:**

当从一个页面向另一个页面发送数据时，数据包括 cookies、post 数据等等。Web 开发人员利用编码技术对黑客隐藏信息。其中一个重要的方法是 base64 编码技术。

**任务:**

> **网站常用的编码类型是什么？**

base64

给出的哈希是**202 CB 962 AC 59075 b 964 b 07152d 234 b 70**，其格式是 md5 哈希。对散列解码后，我们将得到结果 123。
