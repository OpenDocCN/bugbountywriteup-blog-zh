准备—安装netcat (netcat-openbsd)

为了帮助我们制作一个连接监听器，请在 Termux 中安装netcat。

pkg install netcat-openbsd

确保你的手机和受害者的笔记本电脑有相同的网络。然后，检查android的IP地址。

ifconfig

我们的准备工作已经完成。

2。创建Arduino脚本

1。按Windows键。
2。键入“终端”并按回车键，打开终端。
3。键入有效负载以连接到我的termux
4。按回车键
