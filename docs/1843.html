防火墙101:

防火墙是一种负责过滤流量以决定是否允许的设备。如果流量是真正的防火墙允许这些流量，否则拒绝它。

过滤已经通过以下方式完成:

  1. 交通是从哪里来的？
  2. 交通要去哪里？
  3. 这是哪个港口的交通？
  4. 流量使用什么协议？

防火墙的两个主要类别:

  1. 有状态-使用来自连接的全部信息。它决定连接的整体行为，而不是检查单个数据包。
  2. 无状态-检查单个数据包是否被防火墙接受。

VPN基础知识:

VPN代表虚拟专用网络，是一种通过互联网(隧道)在彼此之间创建专用路径连接，在安全的专用网络上进行通信的技术。

VPN的好处:

  1. 允许不同地理位置网络相互连接。
  2. 产生隐私。
  3. 生成匿名。

VPN技术:

  1. PPP-PPTP使用它来进行身份验证并提供数据加密。它不能自己离开网络。
  2. PPTP——它代表点对点隧道协议，允许来自PPP的数据传输并离开网络。它是弱加密的。
  3. IPSec -互联网协议安全使用现有的互联网协议框架加密数据。
