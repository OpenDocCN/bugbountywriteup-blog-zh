# **必备知识**

开始黑客，你应该有一些先决知识。

1.  基本的 Linux 和 Windows(【https://linuxjourney.com】T2
2.  操作系统
3.  建立工作关系网
4.  HTTP、TCP/IP 等基础知识。
5.  一些基本的 python 和脚本
6.  常见漏洞和利用
7.  恶意软件和漏洞的类型
8.  像 PHP、javascript 等网络技术。
9.  精通至少一门编程语言。它可以通过 python、Java、Ruby 等。
10.  学习密码学
