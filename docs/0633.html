学习利用易受攻击的媒体服务器并获得根用户访问权限

利用基于操作系统的漏洞

在我之前的 文章 中，我们讨论了如何利用基于操作系统的漏洞，并通过使用不同的工具和利用后方法提升权限来获得完全的系统访问权限。 冰 是 蓝 室在 TryHackMe 平台上的续作。在本演练中，我们将利用一个安全性很差的媒体服务器，并找到一种方法来获得对windows计算机的完全访问权限。

目标:

  1. 扫描机器以找出打开的端口和针对它们运行的服务
  2. 找到可被利用来获取访问权限的漏洞
  3. 通过破解哈希来收集额外的凭据
  4. 使用post漏洞提升权限

学习成果

  1. 了解黑客周期
  2. 使用OS-INT进行侦察和信息收集
  3. 使用不同的方法和漏洞来危害系统
  4. 获得不同的工具，如 Nmap 、 Metasploit 、 Mimikatz 等，以扫描、利用和收集针对系统的凭证

入门指南

[任务1]连接

首先，我们需要使用OpenVPN连接到TryHackMe网络。这个过程非常简单，你只需要下载配置并在终端上使用 OpenVPN 命令运行它。完成后，在终端上使用 if config 命令验证您是否在TryHackMe的网络上，您应该会看到一个名为“tun0”或“tun1”的接口以及一个分配给它的IP。祝贺您，您现在已经连接到TryHackMe网络。
