任务1-根除它！：

Nmap扫描:

nmap-sC-sV-p--在nmap/ignite上

-sC:默认脚本
-sV:版本检测
-p-:扫描所有端口
-oN:输出将存储在您之前创建的目录“nmap”中

有1个端口打开:
80/http- Apache httpd 2.4.18

Gobuster:

gobuster dir-u http://-w-o-x

-u : URL
-w:单词列表
-o:输出将存储在目录
-x:搜索扩展名，例如html、txt、php、phtml等。
