任务2-损坏机器

Gobuster :

我喜欢这个工具，我经常在CTF使用它。这个工具将帮助我们寻找隐藏的目录。

gobuster dir-u http://-w

  • -u : URL
  • -w:单词表

此外，您可以在gobuster中使用更多标志:

  • -问:安静，无声扫描。将隐藏横幅。
  • -o:要存储在目录中的输出
  • -x:搜索扩展名，如html、txt、php、phtml等。
