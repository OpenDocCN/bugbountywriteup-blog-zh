crackme0x00b

使用文件命令收集信息

$file crackme0x00b

使用 rabin2 工具收集信息。

$rabin2 -I crackme0x00b

给定的破解我是32位和小端，记下这一点，这将有助于找到破解的方法。

使用字符串破解文件

$strings -e L crackme0x00b

使用xxd破解文件

$xxd crackme0x0b

使用rabin2破解文件

$rabin2 -z crackme0x00b

使用radare2破解文件

$radare2 crackme0x00a
 [0x080483e0]> aaa
 [0x080483e0]> pdf @ main
  • aa:-全部分析。
  • aaa:-用更多信息分析所有内容。
  • pdf:-打印反汇编功能。

wcscmp() 使用*s1和*s2变量s1是指向 w0wgreat 和s2是输入字符串。

使用gdb-peda破解文件

gdb-peda就像是gdb的一个插件，你可以从 GitHub 安装。

$gdb crackme0x00a
>gdb-peda$ disassemble main

反汇编main将显示二进制的主要功能。

gdb-peda$ break *0x080484ce
gdb-peda$ run
gdb-peda$ ni

ni或nexti —作为一条指令执行调用。

gdb-peda$ telescope $edx

telescope视图显示堆栈值，并试图取消堆栈可能指向的指针的引用。

还有一种方法可以做上面事情

gdb-peda$ x/25s $edx

x/ :-显示字节

25:-我们要显示多少个

s:——作为一根琴弦

$edx:-这是指寄存器
